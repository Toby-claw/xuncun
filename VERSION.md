## 版本更新检查清单

每次更新时，需要修改以下位置的版本号：

1. **HTML注释**（顶部）
   - 更新当前版本和版本历史

2. **Meta标签**
   ```html
   <meta name="xuncun-version" content="v2.1.0">
   <meta name="xuncun-build" content="20260306">
   ```

3. **JavaScript变量**
   ```javascript
   const xuncun = {
       version: 'v2.1.0',
       build: '20260306',
       ...
   };
   ```

4. **隐秘版本号**（页面右下角）
   ```html
   <div style="...">v2.1.0</div>
   ```

5. **本文件** - 记录版本历史

## 如何查看版本

### 方法1：查看网页源代码
按 `Ctrl+U` 或右键→查看网页源代码，顶部有ASCII艺术字版本信息

### 方法2：浏览器控制台
按 `F12` 打开控制台，可以看到：
- 彩色版本信息自动输出
- 输入 `xuncun` 查看完整版本对象
- 输入 `xuncun.version` 查看版本号

### 方法3：查找隐秘版本号
仔细看页面右下角，有一个几乎透明的8px小字版本号

### 方法4：查看Meta标签
```javascript
// 在控制台输入
document.querySelector('meta[name="xuncun-version"]').content
```
